# D3.js Interactive Data Visualization Dashboards

## Author
Eesha Emaan  

---

## Project Overview
This project consists of two interactive dashboards built using D3.js, focusing on spatial, temporal, network, and hierarchical visualizations. The dashboards transform large real-world datasets into meaningful insights using interactive and linked visual components.

---

## Objectives
- Build interactive dashboards using D3.js  
- Apply spatial, temporal, and hierarchical visualization techniques  
- Implement linked views and coordinated interactions  
- Perform data preprocessing and transformation  
- Enhance user interaction using brushing, zooming, animation, and tooltips  

---

## Task 1: Spatial & Network Analysis Dashboard
### Dataset:
Global Power Plant Database

### Features:
- Zoomable world map with semantic zoom  
- Force-directed clustering by fuel type  
- Brushable timeline for filtering by year  
- Interactive tooltips and linked filtering  

### Key Insight:
Power plants are unevenly distributed globally, and fuel type clustering reveals clear grouping patterns across regions.

---

## Task 2: Temporal & Hierarchical Dashboard
### Dataset:
Gapminder Dataset

### Features:
- Motion chart (GDP vs Life Expectancy)  
- Animated play/pause timeline  
- Choropleth world map  
- Sunburst hierarchy (World → Continent → Country)  
- Interactive year slider  

### Key Insight:
There is a strong correlation between GDP and life expectancy, and disparities between continents become visible over time.
